# Market Performance & Risk Analysis (Crypto vs Equity)

## Overview
A comparative analysis of cryptocurrency and equity assets, focusing on performance
trends, volatility, and risk interpretation rather than price prediction.

## Problem Statement
Different asset classes behave differently under market conditions. Understanding risk
characteristics is essential for informed decision-making.

## Approach
- Analyzed historical price data for Bitcoin and NVIDIA
- Compared trends, volatility, and correlation
- Used time-series analysis for risk interpretation

## Tools Used
Python, Pandas, Data Visualization

## Outcome
The project provides visual insights into asset behavior to support investment and
risk-awareness discussions.
